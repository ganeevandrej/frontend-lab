# frontend-lab

---

## Этот Репозиторий для практики и исследования реальных frontend-проблем в изоляции

---

## Для чего
- восстановить навык самостоятельной разработки
- тренировать инженерное мышление
- изучать реальные проблемы frontend-разработки
- cобирать практические решения и паттерны
- Не стыдно показать в портфолио

---

### Как устроен
Каждый playground/route посвящён одной конкретной проблеме:
- state management
- performance
- async/networking
- UI complexity
- ccessibility
- rendering
- browser APIs

Поэтому репозиторий сфокусирован не на “готовом продукте”, а на решении отдельных инженерных задач и изучении trade-offs между подходами.
